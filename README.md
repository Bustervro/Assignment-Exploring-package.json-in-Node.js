# Basic Node.js Project

## Overview
This project demonstrates how to set up a Node.js application using npm. It includes dependencies, scripts, versioning, and configuration using package.json.

---

## Setup Instructions
1. Initialize project:
   npm init -y

2. Install dependencies:
   npm install express

3. Install dev dependencies:
   npm install --save-dev jest nodemon

---

## How to Run

Start server:
npm start

Run in development mode:
npm run dev

Run tests:
npm test

---

## Dependencies
- express → used to create the server

## DevDependencies
- jest → testing framework
- nodemon → auto-restarts server during development

---

## Scripts
- start → runs index.js
- dev → runs with nodemon
- test → runs Jest tests
